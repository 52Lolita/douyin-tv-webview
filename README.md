#douyin-tv-webview
🎮 遥控器控制
直接操作
↑ → 上一个视频（W）
↓ → 下一个视频（S）
← → 不喜欢（B）
OK（短按）→ 打开评论 / 点击
菜单
OK（长按）→ 打开功能菜单

菜单包含：

鼠标模式
播放器全屏（H）
沉浸模式（Y）
播放 / 暂停（空格）
刷新（Ctrl + R）
自动连播（K）
说明
所有操作通过 模拟键盘事件 实现
鼠标模式下可用方向键控制光标点击网页（主要用于登录和选择推荐菜单栏目）
